## Getting Started

### Initial Set-up

Set up workspace and necessary packages.

* Create a new folder in local machine. This will be the root folder storing both the front-end and back-end of the project (which are maintained in separate repos due to complexity).
	```sh
	mkdir <WORKSPACE-NAME>
	```

* Clone the front-end repository.
	```sh
	git clone <REPOSITORY-LINK>
	```
* Clone the back-end repository
  
* Install all necessary packages
	```sh
	npm install
	```

### Installation

1. Get a free API Key at [https://example.com](https://example.com)
2. Clone the repo
   ```sh
   git clone https://github.com/github_username/repo_name.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your API in `config.js`
   ```js
   const API_KEY = 'ENTER YOUR API';
   ```
